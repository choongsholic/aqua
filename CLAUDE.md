# aqua 도구 위치 (필독 — 과대평가 금지)

aqua는 **마케터가 매주 전월 데이터를 가볍게 확인**하기 위한 **반쪽 도구**. 다음을 혼동하지 말 것:

- **실시간 지표 모니터링은 aqua가 아니다** — 별도 도구가 있고, 업무망도 아닌 **관리망**(더 엄격한 보안)에서만 접속
- aqua는 **업무망**에서 접근 가능한 가벼운 확인용 — 보안상 매주 CSV 수동 등록 방식
- 0.x 시리즈 = 단순 확인에 충실한 시기. **1.0에서 기능 고도화 예정**

## 작업 시 사고 기준
- "실시간 alert", "캠페인 자동 추천", "12주 추세 분석" 같은 무거운 기능 제안 자제 — 도구 역할 초과
- 빨간 테두리 같은 단순 시각 신호는 가벼운 확인 도구에 적합 — alert fatigue 걱정은 과한 분석
- 큰 재설계보다 **기존 톤·구조 유지하면서 다듬기**가 0.x 단계의 정답
- 고도화 아이디어는 1.0 논의에서 꺼내기

# 데이터 구조 (필독 — 추측 금지)

aqua 작업 시작 전에 **반드시** 아래 CSV를 먼저 읽고 세그/트랙 구조를 확인할 것. 코드 grep으로 추정해서 세그명을 만들거나 새로 정의하지 말 것.

## 트랙·세그 (3+3 = 6개, 고정)

**Acquisition Track** (신규/잠재 — 비활성)
- `앱미경험` — 로그인이력 없음
- `장기미이용` — 4개월 이상 연속 미로그인
- `간헐이용` — 그 외

**Retention Track** (활성 사용자, LF 모드의 주 분석 대상)
- `Light` — 1~2일
- `Casual` — 3~7일
- `Power` — 8일~

3+3 구조는 마케터·디자이너·PM 누구나 이름만 보고 직관할 수 있도록 의도된 설계. 세그 추가/이름 변경 제안 금지.

## 데이터 위치 (단일 진실 공급원)
- `data/total.csv` — 상단 대시보드 메트릭
- `data/seg_status.csv` — 6세그 인원수/delta (현황판 카드)
- `data/seg_visit.csv` — Retention 3세그의 일수 분포 0~10일+ (LF 모드)
- `data/seg_panel.csv` — 세그 상세(연령/보유카드/결제 등)
- 주차별 백업: `data_backup/`, `data_4주차/`

## 작업 규칙
- 작업 전 CSV 직접 읽기 (Read 툴). grep으로 코드만 보고 단정하지 말 것
- 시안/예시 데이터 만들 때도 CSV 실값 기준 — 가짜 숫자/세그명 사용 금지
- 새 세그명 도입 금지 (Mid/Heavy/Hi-Heavy 등 임의 명명 절대 금지)
- LF 모드 분포 차트는 11버킷(0일, 1일, ..., 9일, 10일+) — `seg_visit.csv` 컬럼 그대로

# 트리거 규칙

## 아쿠아 (DP 폴더 전체 — Document_templete 제외)
- "아쿠아"라고 하면 `./context.md` 파일을 읽고 이전 작업 맥락을 파악한 뒤 바로 작업 준비 상태로 시작
- "아쿠아 푸시"라고 하면 변경된 파일을 확인하고 `context.md`를 현재 작업 상태로 업데이트한 뒤 git add + commit + push (단, `Document_templete/` 변경분은 제외)
- "아쿠아 풀"이라고 하면 git pull로 최신 파일을 받아오고 완료 여부를 알려줌

## 템플릿 (Document_templete/ 폴더만)
- "템플릿"이라고 하면 `./Document_templete/` 폴더의 context.md(없으면 DP context.md 템플릿 섹션)를 읽고 작업 준비 상태로 시작
- "템플릿 푸시"라고 하면 변경된 파일을 확인하고 context.md 템플릿 섹션을 업데이트한 뒤 `Document_templete/` 내 변경 파일만 git add + commit + push
- "템플릿 풀"이라고 하면 git pull로 최신 파일을 받아오고 완료 여부를 알려줌

# 폰트 렌더링 규칙

## 폰트 스택 (모든 인라인 스타일 / SVG에 적용)
- `'SF Pro Display',-apple-system,BlinkMacSystemFont,'Apple SD Gothic Neo',sans-serif`
- **이유**: 인라인 font: 단축속성은 body font-family를 덮어씀. Apple SD Gothic Neo 없으면 한글이 엉뚱한 폰트로 폴백됨

## SVG 텍스트
- 공백 있는 폰트명은 반드시 따옴표 감쌀 것: `font-family="'SF Pro Display',..."`

## 브라우저 확인
- 수정 후 반드시 **새 탭**으로 열 것 (기존 탭 새로고침은 캐시 문제 있음)
