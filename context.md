# DP-AQUA 프로젝트 컨텍스트

## 프로젝트 개요
현대카드 앱 MAU 800만 달성을 위한 유저 행동 분석 시각화 대시보드.
단일 HTML 파일(`dp-aqua.html`)로 구현된 Canvas 2D 기반 실시간 파티클 시뮬레이션.

## 메인 작업 파일
- `dp-aqua.html` — 메인 대시보드 파일
- `dp-aqua-guide.md` — 구현 가이드 (전체 스펙 정리)

---

## 최근 완료 작업

### 상세페이지 — 주간 UV 추이 차트
- **진입**: 헤더 MAU 영역 클릭 (투명 div 핫스팟, cursor:pointer) 또는 T1 손가락 흔들기 제스처
- **레이아웃**: 좌우 74px 마진 (← 백버튼 텍스트 그리드 기준), 상단 110px, 스크롤 가능
- **KPI**: 당월진행 763만 / 전월동기 749만 / 전월동기대비 ▲14만(green)
- **차트**: 이번달(#50d278) + 전월동기(white 점선), 5주차, Y축 550~850만 / 50만 간격
- **800만 목표선**: green 점선 + "목표→ 800만" 레이블
- **크로스헤어**: 마우스 X → 1일 단위 스냅, 말풍선(4월 N일 + 주차 + 이번달/전월동기)
  - 주차 시작일(1, 8, 15, 22, 29일) = SVG x(80, 375, 670, 965, 1260) 정확히 매핑
  - 틱 피드백: 모바일 진동 4ms / PC Web Audio 클릭음
